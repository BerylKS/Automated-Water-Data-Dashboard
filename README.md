Project Name: Automated Streamflow Dashboard

Description: This tool connects to the USGS Instantaneous Values API to retrieve real-time hydrological data. It processes the raw data using Pandas to clean noise and calculates statistical metrics (mean, max, min discharge). finally, it visualizes the hydrograph using Matplotlib.

Technologies: Python, Pandas, Matplotlib, USGS API.
